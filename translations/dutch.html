Waarom Git Money?

Omdat er iets niet goed is in de wereld. Het enige wat je hoeft te doen is je televisie aan te zetten, je web browser te openen of een krant te lezen. De wereld is gebonden aan de wil van een klein groepje, die beslissingen nemen met het doel te beslissen wat het beste is voor de rest van ons - en het werkt niet. Tot nu toe zijn deze beslissingen uitgelopen op oorlogen, bailouts, inflatie en hogere belastingen. Zonder twijfel zijn er nog veel meer andere negatieve aspecten.

En dat is de reden dat een aantal van ons Git Money hebben gemaakt. Dat is de reden dat we een andere weg gekozen hebben. Maar waarom? Waarom een moeilijker pad kiezen in plaats van de andere dingen te doen? Waarom proberen mensen het Engelse Kanaal over te zwemmen? Mount Everest te beklimmen? Naar gevaar toe te rennen terwijl iedereen wegrent? Wij hebben deze mantra gekozen alsmede onze valuta als bitcoin, niet omdat het makkelijk is, maar omdat het moeilijk is.

We hebben het eerder gezegd en we zeggen het nogmaals - het maakt niet uit waar je vandaan komt of wie je bent. We geloven in het recht om overal vandaan te werken aan wat je maar wilt. Het maakt niet uit wat je geslacht is, je opleiding, je sociale of spirituele geloof of je huidskleur.

Wij zijn het Bitcoin Legioen. Ons thuis is de aarde en we werken in de Cloud - je bent welkom om je bij ons te voegen.

Laat je werk voor zichzelf spreken.

Het hoort niet uit te maken waar je bent - de Verenigde Staten, Oekraïne, een eiland, India of waar dan ook. Als je code een probleem oplost, zou je betaald moeten worden. De kwaliteit van je werk zou het enige moeten zijn dat uitmaakt wanneer je een oplossing biedt voor een probleem. Met git money is je werk je C.V. Verder maakt er niets uit.

Niet een programmeur? Geen probleem.

Een probleem hoeft niet technisch te zijn. Mensen hebben behoefte aan grafisch materiaal, tekst, videobewerking, hulp bij sociale media en vele andere dingen - en ze zijn bereid er voor te betalen. Als je een vaardigheid hebt, wellicht ligt er dan geld op je te wachten.

Hoe het werkt

Vind een bitcoin betalend project en geniet van grenzenloze handel.

git kansen

Doorzoek de huidige bitcoin betalende Kwesties op GitHub. Alle Kwesties met een betaling zullen een groen git money label hebben met "Current Payout:" bij het bekijken. De huidige betaling voor het oplossen van de kwestie is aangegeven in BTC (Bitcoin) en US Dollars. Als je een vraag hebt over de Kwestie, aarzel niet om een reactie te plaatsen en te vragen! Geen ervaring met git?

git werk

Zodra je een kans gevonden hebt, ben je al halverwege thuis (of waar je maar naar onderweg bent!). Ga je gang en ga aan het werk! Het enige wat je hoeft te doen is de repo te klonen, een branch te maken, je aanpassingen in te dienen en een pull request te doen. Zorg dat je pull request bericht vermeldt dat het de kwestie oplost en tevens je bitcoin adres.

git money

Zodra je pull request is gemerged, word je onmiddelijk betaald. Hoe je kiest om git money te gebruiken is aan jezelf. Sommige mensen zoeken naar kleine kwesties die optellen tot een grotere som. Anderen gaan recht op het grote werk af. Misschien ben je er ergens tussen in, kiezend om extra geld bij te verdienen. Het is helemaal aan jou hoe je git money wilt gebruiken.
