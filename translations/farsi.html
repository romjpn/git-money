از هر کجایی که هستید کار کنید

از git money استفاده کنید و برای تمام pull requests هاتون حقوق بگیرید.
بیشتر بدانید

خودتان انتخاب کنید که میخواهید روی چه چیزی کار کنید.

مشکلاتی هستند که نیاز به حل شدن دارند. شما تخصص دارید که اون مشکلات رو حل کنید. این یه فرصت خوبه.

  • بازار آزاد
  • پرداخت اتوماتیک
  • هزینه های به شدت کم
  • کار کردن بدون محدودیت جغرافیایی

اجازه بدید که کار شما بجای خودتان صحبت کند.

هیچ اهمیتی نداره که شما کجا هستین: ایالات متحده آمریکا، اوکراین، یک جزیره، هند، ایران یا هرجای دیگه. اگه کد شما مشکل رو حل میکنه، شما مستحق دریافت حقوق برای اون هستین. وقتی شما راه حلی برای یک مشکل ارائه میدین، کیفیت کار باید تنها چیزی باشه که اهمیت داشته باشه. برای git money کار شما رزومه شماست. چیز دیگه ای اهمیت نداره.

کدنویس نیستید؟ هیچ مشکلی نیست.

یک مشکل که حتما یک مسئله فنی نیست. مردم نیاز به طرح های گرافیکی، نوشتاری، ویرایش و ادیت تصاویر ویدئویی، رسیدگی به رسانه های اجتماعی و خیلی چیزاهای دیگه دارند - و البته حاضر هستن برای حل اون مشکلات پول پرداخت کنند. اگه شما یک مهارت دارید، ممکنه یک پولی منتظر شما باشه.

Git Money چطور کار میکنه

پروژه ای که پرداخت از طریق بیت کوین داره رو پیدا کنید و از تجارت بدون مرز لذت ببرید!

درآمد از git

به محض اینکه pull request شما توی مخزن اصلی ادغام بشه، همون موقع واریزی به شما صورت میگیره. بر همین اساس این شما هستید که نحوه استفاده از git money رو انتخاب میکنید. بعضی ها سراغ issue های کوچک میرن تا با دریافت تعداد زیادی از مبالغ کم پول زیادی بگیرن. بعضی های دیگه هم مستقیما میرن سراغ کارهای بزرگ تر. شاید شما یه جایی بین این دو گروه باشید، با حل یه سری از این مشکلات میتونید یه حقوق جانبی هم داشته باشید. این کاملا به شما بستگی داره که چطور از git money استفاده می کنید.

کار با git

هنگامی که دیدید قابلیت انجام یکی از مشکلات لیست شده رو دارید، نصف راه رو رفتید! حالا باید دست به کار بشید! همه چیزی که لازم دارید این هست که یک clone از اون مخزن (Repo) ایجاد کنید، یک branch بسازید، تغییراتتون رو کامیت کنید و نهایتا یک pull request همراه با یک پیغام توضیح بفرستید برای مخزن اصلی. توی پیامی که همراه pull request میفرستید، مطمئن بشید که ذکر کرده باشید این کدوم مشکل رو حل میکنه و همچنین فراموش نکنید که آدرس کیف پول بیت کوینتون رو هم ذکر کنید. نمیدونید بیت کوین چیه؟ اینجا رو ببینید.

فرصت های git

روی گیتهاب Issue هایی که پرداخت بیت کوین دارند رو باز کنید. تمام Issue هایی که در ازای انجامشون پرداختی صورت خواهد گرفت، دارای یک لیبل سبز رنگ به اسم git money هستند. هنگامی اونها رو باز کنید، شامل قسمتی هستند با عنوان "Current Bounty". پرداختی که در ازای حل کردن این issue انجام میگیره، به BTC (بیت کوین) و معادل دلار آمریکا خواهد بود. اگه سوالی در مورد اون issue دارید، میتونید خیلی راحت همانجا کامنت بذارید و بپرسید. نمیدونید گیت چیه؟ اینجا رو ببینید.
