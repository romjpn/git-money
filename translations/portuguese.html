Como funciona

Encontra um projecto pagável em bitcoin e usufrui de comércio sem fronteiras.

git oportunidades

Procura tarefas renumeradas em bitcoin no GitHub. Todas as tarefas com pagamento têm etiqueta verde "git dinheiro". Dentro de cada tarefa, "Current Bounty:" indica o valor a pagar para a resolução dessa mesma, denominado em BTC (Bitcoin) e dólares americanos. Se tens qualquer questão, não hesites em deixar um comentário na tarefa! Novo para git?

git trabalho

Quando encontrares uma oportunidade, já tens meio caminho andado! Vai adiante e começa a trabalhar! Só tens de fazer clone do repositório, criar uma branch, fazer commit das tuas mudanças e submeter um pull request. Na mensagem da pull request, aponta que estás a resolver o issue e inclui o teu endereço de bitcoin. Novo para Bitcoin?

git dinheiro

Quando o teu pull request for incluído, és imediatamente pago. Tu escolhes a maneira que queres usar git dinheiro. Alguns procuram pequenas tarefas que acumulam a valores maiores. Outros tentam resolver tarefas mais difíceis. Talvez estejas a meio termo, escolhendo tarefas à medida para fazeres uns trocos. Tu é que decides como queres usar git dinheiro.
