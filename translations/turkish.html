Nasıl Çalışıyor?

Bir bitcoin ödemesi yapabileceğiniz proje bulmak, ve sınırsız ticari zevk

git fırsatları

Github üzerinde bitcoin ödemesi alabileceğiniz sorunları görüntüleyin. Bütün sorunlar yeşil etikete sahip ödemeleri içeriyor, " Aktif Görevleri:" görüntüle. Sorunların çözülmesi ile ödemeler BTC (bitcoin) ve Amerikan Doları ile yapılıyor.Sorunlar hakkında sormak istediğiniz sorular varsa sormaktan çekinmeyin! Git'de yeni misiniz?

git çalışması

Bir fırsat bulduğunuzda, belkide evinize gitmek üzereyken (yada herhangi bir yere giderken!). Hemen çalışamalara başlayabilirsin! Bütün yapman gereken bulunan repo'u klonlaman yeni bir branch oluşturman değişiklerini bir mesaj'da toplaman ve pull request'de bulunman.Pull request mesajın içerisinde çözmüş olduğun sorunu notlara eklemen ve aynı şekilde bitcoin adresini eklemen gerekli. git para

Pull request'in kabul edildiğinde, otomatik olarak ödemen yapılmış olacak. Bütün tercih sizin git para'ı seçmiş olmanıza kalmış durumda. Bazı kişilerin küçük sorunlara bakması için belli miktarlar eklemelisiniz. Diğer büyük işler için düzgün gitmesi önemli. Belki sen yolun ortasında ekstra olarak para kazanmak isteyebilirsin. Bunların hepsi senin git para istemenize kalmış durumda.
